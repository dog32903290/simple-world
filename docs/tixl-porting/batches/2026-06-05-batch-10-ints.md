## Batch Notes

- `my_IntListLength` returns `0` for a null or empty list, matching TiXL's null guard.
- `my_IntsToList` maps TiXL's collected integer multi-input to a Vuo integer list port.
- `my_PickIntFromList` preserves TiXL positive modulo wrapping, including negative indices; empty input returns `0`.
- `my_SetIntListValue` preserves `TriggerSet=false` as no mutation, `Index >= 0` as positive modulo selection, `Index == -2` as apply-to-all, and modes `0 Set`, `1 Add`, `2 Multiply`.
- `my_MergeIntLists` preserves Append, HTP, LTP, FailOver, and Average laws within a fixed three-list Vuo adapter. LTP and FailOver state is stored in `nodeInstanceData`.
